# adamjoffe.github.io

## Quickstart

1. Install bundle
```sh
bundle install
```
2. Serve page
```sh
bundle exec jekyll serve
```

3. [Open page](http://localhost:4000)


## Theme - Chirpy

Check out the [theme's docs](https://github.com/cotes2020/jekyll-theme-chirpy/wiki).
